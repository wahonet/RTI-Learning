# 项目合并记录

日期：2026-04-28

`relics-align2` 已合并进 `RTI-Learning`。

原 `RTI-Learning` 偏基础学习，内容包括 RTI、光度立体、采集工具、论文和硬件方案。
原 `relics-align2` 偏实际案例，内容包括碑刻照片增强失败复盘、重新采集路线、数字拓片、汉画像石结构化和 YOLO 入门。

合并后，`RTI-Learning` 作为主仓库，`relics-align2` 作为案例目录保存。

## 共同主题

两个项目都围绕石质文物表面信息采集与整理：

- 碑刻文字
- 汉画像石图像
- 浅浮雕
- 风化和磨损
- 低对比线条
- 榜题和题记

这类问题的关键不是单张照片锐化，而是采集阶段是否留下足够的表面信息。因此技术重点放在多方向斜光、RTI、光度立体、三维模型、法线图、曲率图和数字拓片。

## 原始照片处理

原始碑刻照片和处理结果没有放进 GitHub，原因包括：

- 原图带经纬度水印。
- 文件体积较大。
- 处理图可以通过脚本重新生成。

如需公开数据，需要单独制作脱敏数据集。

## 后续使用方式

1. 主线文档用于 RTI、采集、工具链和数据结构学习。
2. `case-studies` 用于保存具体问题和实验记录。
3. 每个案例记录问题、尝试方法、失败原因、有效资料和后续方案。
4. 采集和标注流程稳定后，再进入模型训练和数据库建设。
