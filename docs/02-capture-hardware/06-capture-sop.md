# 06 RTI与微痕采集SOP草案

## 目标

把RTI理论、工具链复现和微痕成像专题转化为可执行的采集规范。该SOP面向汉画像石、碑刻、浅浮雕、甲骨、刻字石片和自建实验样本，优先支持低成本实验，再逐步升级到固定光源矩阵。

实验记录模板见 `experiments/README.md`。数据集目录和元数据规范见 `datasets/README.md`。质量控制和风险判断见 `docs/05-validation-indexes/14-validation-and-risk-checklist.md`。

## 采集原则

- 非接触：不触碰文物表面，不使用可能污染或损伤文物的材料。
- 固定视角：相机、对象和焦距在一组采集中保持不变。
- 多角度光照：光源方向覆盖上半球，低角度掠射光必须充分采集。
- 可追溯：每组数据记录设备、参数、光照、对象、采集人和处理版本。
- 可重复：采集流程要能被他人复现。

## 方案A：手持/可移动光源法

适用场景：
- 学习RTI基础。
- 野外碑刻快速测试。
- 大型不可移动对象。
- 低成本自采样本。

设备：
- 相机或高质量手机。
- 稳定三脚架。
- 手持LED光源。
- 黑色或银色反光球。
- 比例尺。
- 色卡或灰卡。
- 遥控快门或延时拍摄。

采集步骤：
1. 固定对象和相机，确认构图覆盖目标区域、反光球和比例尺。
2. 设置手动曝光、手动白平衡、手动对焦。
3. 关闭或尽量控制环境光。
4. 从不同方向移动LED光源，每个方向拍一张。
5. 覆盖左右、上下、斜向和低角度掠射光。
6. 每组建议拍摄30-60张。
7. 记录采集距离、光源类型、照片数量和异常情况。

注意事项：
- 不要移动相机。
- 不要让光源进入画面主体区域，除非用于标定。
- 反光球不要过曝。
- 低角度光容易造成强阴影，需要多方向补足。
- 对粗糙石材要避免过强镜面高光。

## 方案B：固定光源矩阵法

适用场景：
- 微痕成像专题复现。
- 小型刻痕样本、甲骨、拓片、硬币、刻字石片。
- 可控实验室环境。

设备：
- 相机或工业相机。
- 固定支架。
- LED环形、弧形或半球光源阵列。
- 控制器，可手动或自动逐个点亮LED。
- 标定板或已知几何灯位。
- 比例尺、色卡、灰卡。

采集步骤：
1. 固定相机、对象和光源矩阵。
2. 标定每个LED的空间位置。
3. 设置相机手动参数。
4. 按顺序逐个点亮LED并拍照。
5. 每张照片记录对应LED编号和光照方向。
6. 采集完成后拍摄一张环境光或暗场参考图。
7. 生成lights.csv或capture.json。

建议光源分布：
- 入门：16-24个光源。
- 标准：32-48个光源。
- 精细：64个以上光源。
- 必须包含低角度光源，适合刻痕显示。

注意事项：
- 光源亮度要尽量一致。
- 光源位置比数量更重要，应覆盖均匀。
- 小样本近距离采集时，要考虑近场光照。
- LED色温要一致，避免颜色变化影响拟合。

## 质量检查

采集后立即检查：
- 图像是否清晰。
- 曝光是否一致。
- 是否有过曝高光。
- 是否有相机或对象移动。
- 是否有足够低角度光照。
- 反光球是否完整可见。
- 文件数量和记录是否对应。

淘汰条件：
- 多张照片明显位移且无法对齐。
- 反光球过曝导致光向无法估计。
- 环境光过强，光源方向效果不明显。
- 目标区域严重反光或阴影遮挡。

质量评分建议：
- 清晰度：线条、刻痕和纹理是否清楚。
- 稳定性：相机、对象和曝光是否稳定。
- 光照覆盖：低角度和高角度是否均衡。
- 可追溯性：元数据和文件名是否完整。
- 研究价值：是否真的帮助识读、线图或标注。

## 面向汉画像石和碑刻的特别规则

- 大对象建议分区采集，每区保留重叠区域。
- 对重要文字和图像局部进行高分辨率补拍。
- 同时拍摄整体环境照片，记录对象位置。
- 风化严重对象要保留原始普通照片和斜光照片用于对比。
- 不要只输出增强图，必须保留原始序列和处理参数。
- 线图和数字拓片必须标明算法生成，不与人工释读混淆。
