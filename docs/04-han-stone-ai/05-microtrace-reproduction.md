# 05 微痕成像复现专题

## 定位

本专题是RTI学习计划的一部分，不是独立主线。目标是在系统掌握RTI、PTM、HSH、RBF、光度立体和工具链之后，吸收公开资料中的“微痕成像”工程路线，用公开可验证的方法搭建复现实验。

边界：
- 只研究公开报道、公开演讲、公开项目、公开论文和通用算法。
- 不反向破解未公开商业算法。
- 不把“微痕成像”直接等同于标准RTI，而是看作RTI同源的多光照/多模态文物成像工程。

## 复现实验路线

### 实验一：RTI到微痕增强

输入：
- 公开RTI数据或自采刻痕照片序列。

处理：
- 生成RTI文件。
- 导出法线图或增强视图。
- 合成低角度掠射光图。
- 做局部对比度增强。

输出：
- 普通照片。
- 斜光照片。
- RTI增强图。
- 法线图。

判断标准：
- 是否能看到普通照片中不明显的刻痕。
- 是否产生伪影。
- 是否可解释增强依据。

### 实验二：数字拓片原型

输入：
- 增强灰度图、法线图或高度估计图。

处理：
- 灰度归一化。
- 背景压制。
- 刻痕方向增强。
- 阈值/反相。
- 小噪点清理。

输出：
- 黑白数字拓片。
- 灰度数字拓片。
- 参数记录。

判断标准：
- 与原物照片相比是否突出刻痕。
- 与传统拓片风格是否接近。
- 是否保留可追溯参数。

### 实验三：线图生成

输入：
- 增强图、法线图、数字拓片。

处理：
- Canny/Sobel边缘检测。
- 结构张量方向估计。
- 形态学闭合和细化。
- 人工校正接口预留。

输出：
- 初步线图。
- 线条置信度图。

判断标准：
- 是否能提取主要笔画和纹样边界。
- 是否误把石材噪声识别为图像线条。

### 实验四：全信息模型草案

输入：
- 原始照片、RTI结果、增强图、拓片、线图、标注。

处理：
- 建立对象档案。
- 记录采集、处理和导出版本。
- 记录标注和释读。

输出：
- JSON/SQLite草案。
- 一个样例对象包。

判断标准：
- 是否能追溯每张输出图来自哪些原始照片和参数。
- 是否能支持后续研究者标注和复核。
