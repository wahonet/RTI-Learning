# Generating archaeological line drawings from limited reference images

来源：[npj Heritage Science, 2026](https://www.nature.com/articles/s40494-026-02526-3)

PDF状态：网页可读；PDF未成功稳定下载。

## 论文在做什么

这篇2026年的论文研究如何在参考图像有限的情况下生成考古线图。作者将任务定义为“文物照片 + 文本提示 -> 二值线图”，使用FLUX.1 Kontext一类扩散Transformer作为基础模型，并通过LoRA进行参数高效微调。

## 方法

核心思路：

- 使用冻结的VAE编码输入文物图像。
- 使用冻结文本编码器编码提示词。
- 在DiT结构中融合图像和文本条件。
- 只训练LoRA层，保留基础模型泛化能力。
- 输出结构更忠实、线条更连续、风格更一致的考古线图。

论文强调普通图像到素描方法在考古图像上容易出问题：表面噪声、密集纹样、病害和复杂背景会导致伪线、断线或过度抽象。

## 对汉画像石的价值

这是“AI线图/数字拓片”方向最新的路线之一。汉画像石样本少、人工线图珍贵，非常符合“有限参考图像”问题。它提示本项目后续可从小规模LoRA微调开始，而不是一开始训练大型模型。

可迁移策略：

- 输入：原图、RTI增强图、法线阴影图、拓片。
- 输出：二值线图或灰度研究线图。
- 条件：对象类型、线条风格、是否保留裂缝/病害、是否只保留图像轮廓。
- 后处理：人工修正，多版本保存。

## 局限

- 论文较新，代码和数据是否公开仍需跟踪。
- 扩散模型生成线图可能“合理但不真实”，对文物释读有风险。
- 必须建立人工审核和来源追踪机制。
