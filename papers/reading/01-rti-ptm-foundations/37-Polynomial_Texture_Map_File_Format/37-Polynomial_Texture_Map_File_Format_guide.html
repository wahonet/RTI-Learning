核心概念逐个拆

核心概念 · file header

文件头，记录格式版本、宽高、类型和关键参数。

核心概念 · PTM_FORMAT

格式标识，告诉软件后续数据怎样解析。

核心概念 · scale/bias

缩放和偏移，用于把模型系数编码到有限数值范围内，再在读取时还原。

核心概念 · LRGB PTM

一种 PTM 表示，把亮度多项式与颜色信息分开，常用于提高效率。

核心概念 · interoperability

互操作性，不同工具能否正确打开和解释同一文件。

核心概念 · 归档风险

格式老旧、软件停更或参数缺失会导致未来无法读取。
