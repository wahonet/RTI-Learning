更自然的中文：这些数字记录和分析方法可以迁移到不同学科，让研究既更快，也能看得更细。

词组拆解：documentation = 记录/文档化，不只是“说明书”；analysis = 分析/释读；adapted to various disciplines = 被不同学科采用并改造。

摘要第3句 · 本文研究对象

贴近原句的直译：本文考察 RTI 方法在碑铭学中为铭文记录和分析带来的创新。

更自然的中文：这篇文章要说明：RTI 怎样帮助碑铭研究者更可靠地记录铭文，并在此基础上分析难读的文字表面。

词组拆解：inscriptions = 铭文；epigraphy = 碑铭学；innovations = 新贡献，不一定是全新算法，也可能是研究流程的改进。

摘要第4句 · RTI定义

贴近原句的直译：RTI 是一种计算摄影方法，它利用从不同角度照亮物体表面所获得的图像数据，合成反射属性，从而记录对象表面的微地形。

更自然的中文：简单说，RTI 会对同一个表面从许多光照方向拍照，再根据明暗变化重建“这个表面哪里凸、哪里凹、朝向哪里”的可视信息。

词组拆解：computational photographic method = 计算摄影方法；reflectance properties = 反射属性；micro-topography = 微地形、微小表面起伏。

摘要第5句 · 为什么适合受损铭文

贴近原句的直译：该方法尤其有助于记录受损考古器物表面，并提高视觉证据的清晰度，从而帮助判断铭文载体上受损表面的字母特征。

更自然的中文：对风化、破损、字口很浅的碑刻来说，RTI 可以让原本不容易看清的刻痕和字形更可见，帮助研究者判断疑字，但最终释读仍要靠专家和证据链。

词组拆解：damaged surface = 受损表面；inscription bearer = 承载铭文的物体，例如石碑、石块、柱础；visual evidence = 视觉证据。

第2页 · 数字记录技术进入文化遗产研究

译读：作者把 RTI 放进数字记录技术谱系：三维激光扫描、摄影测量和 RTI 都在帮助研究者获得更细、更可复查的对象信息。

要点：RTI 不取代 3D，而是补充“表面微地形”和“多光照观察”。

第2页 · 为什么过去不普及

译读：过去数字记录不普及，是因为设备贵、软件复杂、需要专家服务。现在资料、软件和低成本设备更容易获得，研究者可以自己掌握这些方法。

项目解释：这正是本项目先做低成本 H-RTI，再做固定灯阵和光度立体的原因。

第2页 · 风化铭文需要新的证据工具

译读：数字副本可以让研究者在非现场环境中继续观察；而铭文长期受自然和人为因素影响，表面磨损会造成误读甚至无法释读，所以作者引出 RTI 作为新的记录和分析工具。

项目解释：汉画像石榜题、浅刻线、风化纹理也面临同样问题。

第3页 · 传统记录方法及其接触风险

译读：作者回顾拓印、乳胶复制、单色照片、微缩胶片和数字摄影等方法。拓片能得到清晰轮廓，但需要清理表面、湿纸贴合和刷压，对脆弱对象存在接触风险。

要点：RTI 的非接触性，是它进入文物保护流程的核心理由。

第4页 · 单一斜光照片为什么不够

译读：单一方向斜光能显出某些刻痕，却会让其他区域落入阴影；它只能呈现一个光照条件下的表面信息。因此在传统记录不足时，RTI 作为非接触、非破坏、可多方向分析的方法出现。

要点：RTI 的优势是把“观察光源”交给后期交互。

第5页 · PTM/RTI、表面法线与早期应用

译读：PTM/RTI 文件保存像素级表面方向信息，二维图像因此携带微地形线索。作者举苏美尔泥板等早期案例，说明 RTI 能揭示肉眼难辨的文字和表面细节。

数学补课：这里涉及 surface normal、polynomial coefficients、fitting。

第6页 · Dome稳定，H-RTI灵活

译读：固定 dome 能精确控制多光源，适合室内小对象；但造价、体积、野外搬运和供电都是限制。CHI 推动的 H-RTI 用手持光源和反光球降低门槛，更适合现场文化遗产。

项目解释：先掌握 H-RTI，再升级固定灯阵。

第7页 · H-RTI三步流程

译读：RTI 生成包括拍摄、处理、查看三个步骤。通常需要 24 到 60 张多光照照片，相机和对象固定，光源方向变化并被识别。

项目解释：照片序列和灯位信息一旦错配，结果会整体失真。

第8页 · 光照距离、固定要求与现场计划

译读：光源距离通常按对象对角线的三到四倍估计。拍摄期间相机、对象、反光球不能移动，否则像素错位会导致反射计算错误。现场条件复杂，采集前必须规划。

数学直觉：模型假设同一像素跨照片对应同一表面点。

第9页 · RTI与普通图像增强的区别

译读：RTI 在光与表面几何互动明显的对象上最有效。普通照片可以调直方图、锐度和对比度，但它只是在固定光照下加工一张图；RTI 是基于多光照数据重新观察表面。

要点：RTI 是重建观察条件，不只是后期修图。

第10页 · Diffuse Gain 与 Specular Enhancement

译读：RTI Viewer 能对表面法线和颜色值应用数学滤镜。Diffuse Gain 增强相邻像素对比，让浮雕和刻线更明显；Specular Enhancement 增强类似高光的效果，帮助观察细微凹凸。

提醒：增强图用于观察辅助，不能替代原始证据。

第11页 · 案例图与优势讨论

译读：图例显示滤镜如何帮助识别受损字形。作者随后进入讨论：RTI 的主要优势是非接触、非破坏，适合保存状态敏感的铭文。

要点：看清只是第一步，证据可复核才是学术价值。

第12页 · RTI的限制

译读：RTI 只记录表面信息，不能像 X 射线那样看内部结构。它成本低、计算要求低、适合细节观察；但如果需要严格三维测量，应使用激光扫描或摄影测量。

要点：RTI 用于读表面，3D/SfM 用于量几何。

第13页 · 结论：RTI服务释读和证据链

译读：作者认为 RTI 既适合小型出土物，也适合大型不可移动文物。采集后，研究者可以在办公室继续基于数字数据细读对象；交互光照和增强滤镜能帮助检查字形、刻痕和损伤，让铭文释读拥有更多可复核证据。

最终要点：RTI 不替专家判断，而是提高证据质量和复查能力。

译读：这一页进入参考文献。它把论文放进几个来源传统里：CHI 的 RTIBuilder/RTIViewer 指南、装饰技法与器物表面分析、数字替身、以及大型对象的高质量 PTM/RTI 采集。

译读：这一页列出 RTI/PTM 的关键源头：Malzbender 的 Polynomial Texture Maps、实时光度立体与表面增强、安提基特拉机械的交互重打光，以及古希腊罗马硬币的 RTI 研究。

读法：如果你要追 RTI 的技术根，先读 PTM；如果你要追文物应用，硬币、玻璃、楔形文字和碑铭是最相关的案例群。

第1页 · 标题、摘要、关键词

第1页 · 标题段
关键词：Epigrafi / Epigraphical Studies / Documentation / Analysis / RTI

中文意译：论文把 RTI 定位为碑铭学的一种新型“记录 + 分析”方法。它不是单纯为了把照片拍得好看，而是为研究者提供可复核的铭文表面证据。

读法：标题里的 documentation 是“记录证据”，analysis 是“基于证据做释读”。这两个词要分开看：没有稳定记录，后面的释读就无法复核。
第1页 · 土耳其语摘要第1段
关键词：数字成像方法、文化遗产、新方法、快速细致研究

中文意译：作者先说，数字成像技术的发展让文化遗产研究获得了新的记录方式。这些新方法能被不同学科采用，让研究更快、更细，也更容易比较细节。

项目提示：这说明 RTI 不是孤立工具，而是文化遗产数字记录的一部分。你后续做汉画像石时，也要把 RTI 放进摄影测量、3D、拓片、线图、标注这一整套证据链里。
第1页 · 土耳其语摘要第2段
关键词：Reflectance Transformation Imaging、铭文、记录流程、分析流程

中文意译：本文研究 RTI 在碑铭学中的作用，尤其是它怎样改进铭文的记录和分析。作者强调 RTI 已经在文化遗产研究中使用，并且适合处理文字载体表面的可见细节。

这里对应你的项目：碑刻、榜题、汉画像石边缘线条都属于“文字/图像刻在表面上”的问题，RTI 的价值正是把表面起伏变成可反复观察的数字证据。
第1页 · RTI定义段
关键词：多角度照明、反射属性、micro-topography、computational photography

中文意译：RTI 是一种计算摄影方法：对同一对象从不同方向打光拍摄，再把这些图像中的反射变化合成为表面微地形信息。它让人看到肉眼或单张照片不容易看出的细节。

数学直觉：每张照片对应一个光照方向；同一像素在不同光照下明暗变化，反映了局部表面朝向。先看 向量和法线 与 点积和斜光。
第1页 · 英文摘要段
关键词：damaged surfaces / inscription bearer / visual evidence / non-destructive

中文意译：作者强调 RTI 特别适合损坏、风化或难以阅读的铭文表面。它能增强视觉证据，让研究者更好地判断残损字母、刻线和表面形态，同时保持非接触、非破坏。

你要抓住“visual evidence”这个词：RTI 不是替专家做最终释读，而是给专家更多可检查证据。

第2页 · 引言：为什么文化遗产需要数字记录

第2页 · 引言第1段
关键词：3D laser scanning / photogrammetry / RTI / micro-topography

中文意译：作者把 RTI 放在一组数字记录技术里比较：三维激光扫描可以获取精细三维数据，摄影测量可以通过图像匹配重建形体，而 RTI 擅长揭示对象表面的微小起伏。

项目迁移：整体形体用 3D/SfM，局部浅刻用 RTI/光度立体。不要指望一种技术解决所有问题。
第2页 · 引言第2段
关键词：设备复杂、高成本、用户友好、低预算

中文意译：过去数字记录方法不普及，主要因为设备和软件复杂、成本高，需要外部专家或服务商。现在资料更容易获得，软件更友好，低成本设备更多，研究者可以自己尝试。

这正是你项目的路线：先用可负担设备建立方法闭环，再逐步提高灯位、标定和验证精度。
第2页 · 引言第3段
关键词：virtual conservation / digital copy / visual perception

中文意译：数字记录可以形成文物的数字替身，用于保存、展示和分析。通过增强视觉感知，研究者可以在办公室继续查看已经记录下来的细节。

注意“数字替身”不是替代原物。它的价值是可访问、可传播、可复查，但仍要保留采集条件和处理参数。
第2页 · 引言第4段
关键词：自然/人为磨损、误读、无法释读

中文意译：铭文表面会因为自然过程和人为因素被磨损，这会让文字释读变得困难。有些铭文会被误读，有些甚至无法读出。论文因此提出：需要一种新的记录和分析工具。

项目提示：汉画像石同样有风化、拓片残缺、线条浅、背景噪声等问题。RTI 的任务是降低“看不见”的风险，而不是自动给出唯一答案。

第3页 · 传统碑铭记录方法与限制

第3页 · 传统记录段
关键词：estampage / latex / photograph / inventory

中文意译：作者回顾了碑铭记录方法：19-20世纪常用拓印、乳胶复制等方式，后来摄影、微缩胶片和数字摄影成为记录和编目的常见手段。

这里不是否定传统方法，而是在说明每种记录方式都有历史用途和局限。RTI 应该作为补充证据，而不是简单替代拓片。
第3页 · 拓片操作段
关键词：清理表面、湿纸、刷压、接触对象

中文意译：拓片需要清理铭文表面，把特制纸张贴合到刻痕里，再通过刷压让纸进入凹槽。这能得到清晰的文字轮廓，但它必须接触文物表面。

项目提示：对脆弱对象，接触式方法有风险。RTI 的一个核心优势就是不接触表面。
第3页 · 摄影限制段
关键词：单一光照、阴影、反光、细节不可见

中文意译：普通照片取决于拍摄时的光照角度。某些刻线会被阴影突出，另一些细节会被阴影或反光遮住。单张照片很难同时展示所有表面信息。

这就是你截图里左边 PDF 看不清的更大版本：普通图像把观察条件固定死了，RTI 把观察光源交还给读者。

第4-6页 · RTI是什么，以及为什么 H-RTI 变得实用

第4页 · 转入RTI段
关键词：alternative method / non-destructive / multi-directional analysis

中文意译：当传统方法不足以记录受损、难读或保存状态敏感的铭文时，RTI 成为一种替代方案。它不接触文物，并允许从多个方向分析表面。

重点：RTI 的“多方向”不只是拍摄时多方向打光，也是阅读时可多方向重打光。
第4-5页 · PTM/RTI技术解释段
关键词：Polynomial Texture Map / surface normals / coefficients

中文意译：论文解释 RTI 与 PTM 的技术背景：对每个像素记录在不同光照下的亮度变化，用数学模型保存这种变化。模型里包含表面法线等信息，因此二维图像可以携带表面形态线索。

补课：PTM 会出现 polynomial、coefficients、surface normal。对应 最小二乘与拟合。
第5页 · 文化遗产应用史段
关键词：Sumerian tablet / cultural heritage / surface details

中文意译：作者提到 RTI 早期应用在铭文对象上，例如苏美尔泥板，研究者通过 PTM/RTI 看到了肉眼不容易分辨的文字和表面细节。

这说明 RTI 从一开始就和“难读文字”关系密切。汉画像石榜题、碑刻疑字可以沿这条线走。
第6页 · dome系统限制段
关键词：dome / high cost / small objects / indoor use

中文意译：固定半球 dome 可以控制灯位，但造价高、体积受限，不适合野外大型对象。它更适合室内小件，比如硬币、化石等。

这段对硬件路线很重要：固定 dome 稳定，手持 H-RTI 灵活。项目要根据对象大小选择。
第6页 · CHI与H-RTI段
关键词：Cultural Heritage Imaging / highlight method / black reflective spheres

中文意译：为了降低成本并适应现场条件，CHI 等推动了 Highlight RTI：相机固定，手持光源移动，通过画面中的黑色反光球估计每张照片的光照方向。

这是你做现场碑刻和画像石最应先掌握的方法，因为它不要求一开始就有大型灯阵。

第7-8页 · H-RTI采集流程

第7页 · 三个步骤段
关键词：photo capture / processing / viewing mathematical model

中文意译：作者把 RTI 生成分成三步：拍摄对象的一组照片；处理照片生成模型；在查看器中交互查看这个数学模型。

项目记录也要按这三步建目录：capture、processing、viewing/export。不要把原图、结果图和说明混在一起。
第7页 · 24-60张照片段
关键词：24-60 photos / fixed camera / changing known light angles

中文意译：H-RTI 通常对同一对象拍 24 到 60 张照片。相机固定，对象固定，光源方向变化且需要被估计或记录。

实操重点：每张图的文件名和光照顺序要严格保存；少几张可以补，顺序错乱会毁掉后续处理。
第8页 · 光源距离段
关键词：object diagonal / three or four times / light distance

中文意译：论文提到，光源到对象的距离可按对象对角线长度的三到四倍来估计。这样能让光照近似为来自远处的方向光，减少近场误差。

数学直觉：如果灯太近，不同像素看到的光照方向差异会变大，简单 RTI 模型会变不稳。
第8页 · 固定相机和反光球段
关键词：camera fixed / spheres fixed / pixel shift / wrong reflectance calculations

中文意译：采集过程中相机和反光球必须保持不动。哪怕很小的像素位移，也会导致错误的反射计算，结果可能变模糊、不真实，无法用于分析。

这段应该直接进入你的采集 SOP：三脚架、防抖、遥控快门、标尺、反光球固定都不是形式主义。
第8页 · 现场计划段
关键词：field conditions / preparation / planning

中文意译：作者强调现场采集前的准备和计划非常重要。野外对象受空间、天气、光线、对象高度、可接近性影响，不能到现场才临时决定怎么拍。

项目提示：每次采集前先填 capture plan：对象尺寸、相机位置、光源半径、反光球位置、预计张数、环境光控制。

第9-10页 · 交互查看与滤镜

第9页 · RTI适用场景段
关键词：light interaction / geometry / reflectance / surface texture

中文意译：RTI 在光与表面几何互动明显的对象上最有效。材料反射性质和表面纹理会给分析提供感知线索。

浅刻、凿痕、风化裂隙适合 RTI；平滑无起伏、颜色信息为主的对象，RTI价值会下降。
第9页 · 普通图像增强限制段
关键词：histogram / sharpness / contrast / ordinary photo editing

中文意译：普通照片也可以调直方图、锐化、增强对比度，但这些操作只是处理固定光照下的一张图，不能像 RTI 那样重新选择光照方向。

这能解释“P图”和“RTI增强”的区别：前者增强现有图，后者利用多光照数据重新观察表面。
第10页 · RTI Viewer滤镜段
关键词：surface normals / color values / mathematical transformation filters

中文意译：RTI Viewer 可以对表面法线和颜色值应用数学变换滤镜，从而突出难以察觉的表面细节。

数学直觉：滤镜不是凭空创造信息，而是基于多光照估计出的表面方向和颜色数据重新映射视觉效果。
第10页 · Diffuse Gain段
关键词：diffuse gain / contrast between neighboring pixels / relief / incision

中文意译：Diffuse Gain 通过增强相邻像素之间的对比，让浮雕和刻线更明显。它适合看字口、凹槽、浅刻边缘。

读碑刻时，Diffuse Gain 常常比普通视图更能暴露笔画方向和残损边缘。
第10页 · Specular Enhancement段
关键词：specular enhancement / reflective contribution / surface detail

中文意译：Specular Enhancement 通过增强类似高光的视觉效果，让表面起伏更容易被看出来。它适合观察细微凹凸，但也可能让图像看起来不自然。

项目规则：增强图可用于观察和标注辅助，但不要把它当作原始证据；必须能回到原图和采集参数。

第11-12页 · 优势与限制

第11页 · 优势第1段
关键词：non-destructive / contactless / photographic capture

中文意译：RTI 是非破坏、非接触的记录方法。它通过拍照获得表面细节，避免对脆弱铭文进行拓印或其他接触式操作。

这对馆藏和风化现场对象尤其重要：安全性本身就是方法价值。
第11页 · 优势第2段
关键词：portable equipment / low cost / field use

中文意译：H-RTI 所需设备相对便携且成本较低，适合野外或远距离地点。这让研究者不必依赖大型设备也能做表面记录。

这支持项目第一阶段：先建立手持 H-RTI 能力，再考虑固定灯阵和定量光度立体。
第12页 · 限制第1段
关键词：surface data only / no subsurface / not X-Ray

中文意译：RTI 只能记录表面可见信息，不能像 X 射线那样分析内部或下层结构。如果信息不在表面，RTI无法提供答案。

项目提示：不要把 RTI 说成万能修复。它能增强可见表面证据，不能恢复不存在或被完全覆盖的信息。
第12页 · 限制第2段
关键词：3D laser scanning / high cost / metric data

中文意译：三维激光扫描可以提供精确几何和测量数据，但成本更高。RTI 文件生成和查看不需要很高配置，却不能提供严格三维度量。

结论：如果目标是“看清刻线”，RTI 很合适；如果目标是“精确测量三维尺寸”，要用摄影测量、激光扫描或光度立体等方法补充。
第12页 · 限制第3段
关键词：surface shape / interpretation / verification

中文意译：RTI 可以帮助理解表面形态，但解释仍然需要研究者判断。增强滤镜可能突出细节，也可能放大噪声或误导初学者。

所以你的 HTML 导读需要“解释”和“风险提醒”并列，不应该只给一个结论。

第13页 · 结论：RTI怎样服务碑铭学

第13页 · 结论第1段
关键词：small finds / immovable heritage / correct documentation

中文意译：作者认为，无论是小型出土物还是大型不可移动文物，RTI 都能帮助做更准确、更细致的数字记录。

汉画像石正好处在这里：既可能有馆藏小件，也可能有大型不可移动石刻。
第13页 · 结论第2段
关键词：office analysis / digital data / detailed study

中文意译：采集后的 RTI 数据可以带回办公室继续分析。研究者不必每次都回到现场，也能从不同光照方向反复检查细节。

这就是“可复核材料”的意义：论文、报告、标注、释文都可以回指到同一个 RTI 数据包。
第13页 · 结论第3段
关键词：decipherment / characters / macro and micro detail

中文意译：在释读铭文时，RTI 的交互光照和增强滤镜可以帮助研究者从宏观和微观两个层级查看字形、刻痕和表面损伤。

项目迁移：榜题识读可以设置多层证据：原图、RTI普通视图、Diffuse Gain、Specular Enhancement、人工释读。
第13页 · 结论第4段
关键词：low cost / non-destructive / epigraphy

中文意译：作者最终强调，RTI 因低成本、便携、非接触和分析能力，适合作为碑铭学研究中的重要数字方法。

一句话收束：这篇论文证明 RTI 对碑铭学的价值不是“图像漂亮”，而是“证据更丰富、解释更可复核”。

读完后你要记住的结构

  • 传统记录：拓片、乳胶、摄影都有价值，但都有局限。
  • RTI机制：多光照照片记录表面反射变化，支持交互重打光。
  • H-RTI优势：便携、低成本、适合现场和不可移动对象。
  • RTI限制：不是真实3D测量，不看内部结构，不替代专家释读。
  • 项目落地：RTI结果要进入证据链，而不是只当展示图。
