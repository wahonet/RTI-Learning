# RTI 论文阅读库

本目录已经按“技术栈 + 阅读顺序 + 单篇论文文件夹”整理。

## 目录规则

- 每个技术栈一个一级文件夹，例如 `01-rti-ptm-foundations`、`04-photometric-stereo`。
- 每篇论文一个文件夹，文件夹名前保留阅读序号，例如 `07-RTI_Epigraphical_Studies`。
- 同一篇论文的 PDF、HTML 导读、JSON 联动数据、Markdown 笔记尽量放在同一文件夹。
- `interactive-reader.html` 是通用 PDF 精确联动阅读器，`vendor/` 是它使用的 PDF.js 依赖。

入口：`index.html`。
