# Automated generation of archeological line drawings from sculpture point cloud based on weighted centroid projection

来源：[npj Heritage Science, 2025](https://www.nature.com/articles/s40494-025-01678-y)

## 论文在做什么

这篇论文研究如何从石雕点云中自动生成考古线图。传统考古线图依赖人工绘制，耗时且主观性强。作者的目标是从带噪、无组织点云中提取特征线，形成连续、平滑、可用于发表和测量的三维线图。

## 方法

方法分为四步：

- 特征点检测：使用邻域加权质心沿法线方向的投影作为几何变化度量。
- 特征点细化：把检测点投影到真实特征线附近，提升线的位置一致性。
- 边界点检测：用角度准则识别边界点。
- 曲线生长：综合距离和角度因素连接特征点与边界点，生成完整线图。

加权质心使用两个权重：

- 表面粗糙度。
- 法线差异。

这使方法对点云噪声更稳健，也能区分凹线和凸线。

## 对汉画像石的价值

汉画像石如果有结构光、激光扫描、摄影测量或光度立体积分出的点云/高度图，就可以走“几何线图”路线，而不只依赖二维图像边缘。对浅浮雕、减地平雕、石雕轮廓、边框纹样尤其有价值。

可迁移方向：

- 从3D扫描点云提取宏观轮廓。
- 从光度立体法线/高度图提取局部微痕线。
- 把二维AI线图和三维几何特征线互相验证。
- 在IIML中区分`trace`、`edge`、`reliefBoundary`和`interpretiveLine`。

## 局限

- 论文主要面向石雕点云，不是二维照片或RTI文件。
- 汉画像石很多对象只有拓片或普通照片，无法直接使用点云方法。
- 点云质量、法线估计和采样密度会强烈影响线图。
- 自动线图仍不能判断图像学关系，比如人物动作、器物属性和叙事场景。
