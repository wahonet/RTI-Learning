# RTI Learning

这是一个面向 RTI、PTM、光度立体、三维采集、数字拓片、碑刻释读和汉画像石结构化的学习与研究资料库。

## 现在怎么读

1. 先打开 [docs/README.md](docs/README.md)，看项目文档分组。
2. 论文从 [papers/reading/index.html](papers/reading/index.html) 进入。那里已经按技术栈整理，并把同一篇论文的 PDF、HTML 导读、JSON 联动数据和 Markdown 笔记放在同一文件夹。
3. 数学基础看 [math-foundations/html/index.html](math-foundations/html/index.html)。
4. 案例复盘看 [case-studies/stele-clarity-2026-04-27](case-studies/stele-clarity-2026-04-27)。

## 顶层目录

- `docs/`：项目路线、理论、采集、硬件、软件、汉画像石 AI、验证清单和索引表。
- `papers/reading/`：整理后的论文阅读库。
- `math-foundations/`：高数、线代、优化、光照几何和机器学习中文小课。
- `datasets/`：数据集索引和本地/外部数据说明。
- `experiments/`：实验记录模板。
- `case-studies/`：具体碑刻/汉画像石案例复盘。
- `repositories/`：工具仓库快照和复现入口。
- `tutorials/`：早期学习教程，后续可逐步吸收到 docs。
- `vocabularies/`：汉画像石标注词表。

## 论文目录规则

`papers/reading/技术栈/阅读序号-论文名/`

例如：`papers/reading/02-heritage-epigraphy-cases/07-RTI_Epigraphical_Studies/`。

旧入口 `papers/deep-reading/index.html` 已保留为跳转页，已有浏览器地址不会直接断掉。
