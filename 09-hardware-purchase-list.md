# 09 RTI微痕扫描硬件购买清单

这份清单面向汉画像石、碑刻、浅浮雕、甲骨、刻字石片等对象的RTI和微痕成像实验。目标不是一次买齐所有设备，而是按预算逐步搭建：

```text
低预算：先跑通H-RTI和基础多光照采集
中预算：建立稳定的固定光源矩阵和光度立体实验系统
高预算：建设接近文博/科研级的微痕成像实验平台
```

价格为人民币粗略区间，实际采购会随品牌、成色、渠道和是否已有设备而变化。

## 1. 总体采购原则

- 优先保证稳定性：相机、对象、灯位必须稳定，稳定性比单个设备参数更重要。
- 优先拍RAW：后续做RTI、法线估计和增强处理时，RAW比直接JPEG更可靠。
- 光源要可控：亮度、色温、位置、编号和触发顺序都要可记录。
- 标定件必须买：反光球、灰卡、色卡、比例尺和标定板决定数据是否可复现。
- 不碰文物表面：所有支架、灯架、标尺、色卡都不能直接接触珍贵文物表面。
- 先小样本验证：硬币、刻字石片、手工刻痕板跑通后，再上真实碑刻和汉画像石。

## 2. 低预算方案：学习验证型

预算区间：约3000-10000元。如果已有相机或高质量手机，成本可压到3000-5000元。

适合目标：

- 学习RTI原理。
- 复现Highlight RTI。
- 拍摄硬币、刻字石片、局部碑刻。
- 生成RTI文件、重打光图、基础增强图。

### 采购清单

相机与镜头：

- 优先使用已有微单、单反或高质量手机。
- 如果需要购买，选择二手APS-C微单/单反机身，约2000-4000元。
- 近摄优先买微距镜头，约1500-4000元；预算不足可先用近摄接圈或手机微距夹镜。
- 必须支持手动曝光、手动白平衡、手动对焦，最好支持RAW。

稳定支撑：

- 稳定三脚架，约300-800元。
- 手机/相机快装板、水平仪，约50-200元。
- 遥控快门线或无线快门，约50-200元。
- 小型滑轨或微距调焦台，约200-600元，可选。

照明：

- 高显色LED手电或摄影补光灯2-4个，约300-1000元。
- 柔光纸、遮光布、黑布，约100-300元。
- 小灯架或万向夹，约100-400元。
- 建议色温固定在5000K-5600K附近，不要混用明显不同色温的灯。

标定与记录：

- 黑色反光球2-3个，建议直径10-30mm和40-60mm各一类，约50-300元。
- 灰卡，约50-150元。
- 色卡，入门款约100-500元。
- 毫米比例尺、考古摄影标尺，约50-200元。
- 记录本、标签纸、编号贴，约50-100元。

数据与计算：

- 128GB或256GB高速存储卡，约100-300元。
- 1TB移动SSD，约400-700元。
- 使用现有电脑即可，建议至少16GB内存。

### 能做到什么

这套方案可以完成“从拍摄到RTI查看”的入门闭环：

```text
固定相机
手持光源多角度拍摄
反光球估计光照方向
RelightLab/RTIBuilder处理
RTIViewer/OpenLIME查看
导出增强图
```

### 局限

- 光源方向不够精确，难以做高可靠光度立体。
- 手持灯重复性差，不适合严肃对比实验。
- 对大幅汉画像石只能做局部试验。
- 微痕增强可以观察，但还谈不上稳定计量。

## 3. 中预算方案：固定光源矩阵实验型

预算区间：约30000-80000元。

适合目标：

- 复现“光源矩阵 + 微痕增强”路线。
- 做校准光度立体和法线图。
- 生成虚拟掠射光、数字拓片和初步线图。
- 建立后续软件开发所需的标准数据集。

### 采购清单

相机与镜头：

- 2400万-4500万像素微单或单反机身，约8000-20000元。
- 60mm、90mm或100mm微距镜头，约3000-8000元。
- 稳定电源适配器或假电池，约200-500元。
- 快门线或电脑联机拍摄方案，约200-1000元。

稳定支撑：

- 专业三脚架或翻拍架，约1500-6000元。
- 可调横臂、云台、微调平台，约1000-4000元。
- 小型防震工作台或厚重平台，约1000-5000元。
- 对小样本建议使用翻拍架，对碑刻局部建议使用重型三脚架和横臂。

固定光源矩阵：

- 16-32路高显色LED，约2000-8000元。
- LED驱动、电源、散热件，约1000-3000元。
- Arduino、ESP32、树莓派或工控控制器，约200-1500元。
- 继电器/MOSFET控制板，约200-800元。
- 半球、弧形或环形支架材料，约1000-6000元。
- 灯位编号、线缆管理、接插件，约300-1000元。

标定与测量：

- 高质量黑色反光球和银色镜面球，约200-800元。
- 色卡、灰卡、白平衡卡，约300-1500元。
- 棋盘格标定板、ArUco/AprilTag标定板，约100-500元。
- 游标卡尺、激光测距仪、小型水平仪，约300-1000元。
- 标准刻线样本或微细纹理测试片，约500-3000元。

环境控制：

- 遮光棚、黑布、挡光板，约500-2000元。
- 恒定色温辅助灯，可选，约500-2000元。
- 防尘罩、样本盒、无酸纸、手套，约300-1000元。

数据与计算：

- 2TB-4TB移动SSD或NAS基础存储，约1000-4000元。
- 处理电脑建议32GB-64GB内存，独立显卡可选，约8000-20000元。
- 如果已有高性能电脑，可只补充存储。

### 推荐优先采购顺序

1. 相机、微距镜头、稳定支架。
2. 反光球、色卡、灰卡、比例尺。
3. 16路固定LED矩阵。
4. 控制器和自动逐灯拍摄脚本。
5. 存储与备份设备。
6. 升级到32路或48路光源。

### 能做到什么

这套方案能进入真正的微痕实验：

```text
固定灯位
逐灯拍摄
记录lights.csv
估计法线图
合成虚拟掠射光
生成增强灰度图
生成数字拓片初稿
```

### 局限

- 光源矩阵需要自己设计、布线、标定和维护。
- 对大幅不可移动文物，需要分区采集或移动式支架。
- 如果要做计量级高度恢复，还需要更严格的几何标定和真实标样。

## 4. 高预算方案：科研/文博平台型

预算区间：约150000-600000元以上。

适合目标：

- 建设长期科研平台。
- 支持汉画像石、碑刻、甲骨、金属器、陶器等多类对象。
- 兼顾RTI、光度立体、偏振、多光谱和3D融合。
- 形成可交付给文博单位或课题组的标准化采集系统。

专业光源系统：

- 可编程控制系统，约5000-30000元。

## 5. 三档方案对比

低预算方案适合“学会RTI是什么”。它的核心是相机、三脚架、手持LED和反光球，优先跑通采集和处理流程。

中预算方案适合“开始复现微痕成像”。它的核心是固定光源矩阵、微距成像、灯位标定和光度立体，是我们近期最推荐的主线。

高预算方案适合“建设长期平台”。它的核心是高分辨率成像、可编程多光源、偏振/多光谱/3D扩展和稳定的数据管理。

## 6. 最推荐的采购路线

如果我们从零开始，建议不要直接买高预算全套，而是分三步：

### 第一步：低预算H-RTI包

先买：

- 稳定三脚架或翻拍架。
- 高显色LED手电/补光灯。
- 黑色反光球。
- 灰卡、色卡、比例尺。
- 移动SSD。

如果已有相机，这一步就能马上开始。

### 第二步：中预算固定光源矩阵

再买：

- 微距镜头。
- 16路LED矩阵。
- 控制器。
- 翻拍架或重型支架。
- 标定板。

这一阶段是微痕成像的关键。

### 第三步：扩展到科研平台

最后根据实验结果决定是否加：

- 32-96路光源。
- 偏振相机。
- 近红外/多光谱。
- 3D扫描。
- 高性能工作站和NAS。

## 7. 第一批实际购买建议

如果现在就要下单，建议先买中低预算组合，而不是低预算纯入门：

```text
已有相机：
  预算约8000-20000元
  买微距镜头、三脚架/翻拍架、LED、反光球、色卡、比例尺、SSD、基础控制器

没有相机：
  预算约18000-40000元
  买APS-C/全画幅机身、微距镜头、三脚架/翻拍架、LED、标定件、SSD

准备做固定矩阵：
  预算约30000-80000元
  在上面基础上加入16-32路LED矩阵、控制器、支架和工作站/存储
```

## 8. 需要避免的采购误区

- 不要先买昂贵相机却用不稳定灯光，RTI质量主要受光照和稳定性影响。
- 不要混用不同色温、不同亮度、不同扩散特性的LED。
- 不要只买环形灯，普通环形灯不能替代RTI需要的多方向逐灯照明。
- 不要忽视标定件，没有反光球、灰卡、色卡和比例尺，数据很难复核。
- 不要直接承诺“微米级”，除非有真实标样和计量设备验证。
- 不要把增强图当原始证据，必须保留原始照片和处理参数。
