# 本地下载教材说明

本目录用于存放开放教材 PDF，默认不提交到 Git。`.gitignore` 已忽略本目录下的 PDF，只保留本说明文件。

已下载或建议下载的资料：

- `openstax-calculus-volume-1.pdf`：OpenStax Calculus Volume 1，函数、极限、导数、积分基础。
- `openstax-calculus-volume-2.pdf`：OpenStax Calculus Volume 2，积分技巧、级数、参数方程、极坐标等。
- `openstax-calculus-volume-3.pdf`：OpenStax Calculus Volume 3，向量、多元函数、偏导、梯度、重积分。
- `boyd-vandenberghe-introduction-to-applied-linear-algebra.pdf`：向量、矩阵、最小二乘、线性模型。
- `boyd-vandenberghe-convex-optimization.pdf`：凸优化、最小二乘、约束优化。
- `mathematics-for-machine-learning.pdf`：线性代数、解析几何、矩阵分解、概率、优化基础。

这些 PDF 仅供本地学习使用。若要公开仓库，请不要提交 `downloads/` 中的大文件。
