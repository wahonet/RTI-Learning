RTI/PTM 与汉画像石数字研究学习路线

这是一份面向个人学习和后续实验复现的总路线图。目标不是只学会RTIViewer里拖动光源，而是建立从多光照采集、RTI/PTM/HSH/RBF、光度立体、3D融合、数字拓片、YOLO/SAM标注，到IIML/图像学知识库的完整技术栈。

7学习阶段：基础、案例、汉画像石、光度立体、前沿、AI标注、知识库
34+核心论文和资料条目，含经典PTM、RTI、光度立体和2024-2026新论文
17+仓库与工具入口：Relight、OpenLIME、NeuralRTI、CVAT、AnyLabeling等
3档硬件路线：低预算H-RTI、中预算固定光源矩阵、高预算科研平台

核心技术闭环

所有学习和采购都围绕这条闭环展开。最重要的原则是保留原始多光照图像MLIC，后续所有新算法都可以从MLIC再出发。

采集RAW照片、多方向光照、反光球、灰卡、比例尺、采集元数据。
拟合PTM、HSH、RBF、Relight格式、法线图、平均图、增强图。
重建光度立体法线/高度、SfM整体模型、V-RTI、局部RTI配准。
转译数字拓片、研究线图、点云特征线、AI候选轮廓。
结构化YOLO/SAM标注、IIML、图像学关系、知识图谱和Web发布。
第一阶段不要追求所有前沿技术同时落地。最稳妥的路线是：H-RTI跑通、固定光源矩阵、光度立体基线、小样本YOLO/SAM/IIML，再进入NeuralRTI、多光谱、偏振和扩散线图。

需要储备的基础知识

这部分决定你能不能判断结果是否可信。RTI项目失败通常不是软件不会用，而是摄影、光照、标定、数据管理和文物解释层没有打通。

摄影与采集

  • 手动曝光、手动对焦、RAW、白平衡、景深、微距。
  • 相机固定、对象固定、镜头畸变、快门震动控制。
  • 灰卡、色卡、比例尺、反光球、文件命名。

光照与物理

  • 入射光方向、掠射光、漫反射、镜面反射、阴影。
  • PTM/HSH/RBF如何拟合光照函数。
  • 光度立体如何从亮度恢复法线和高度。

数学与图像处理

  • 向量、法线、坐标系、矩阵、最小二乘。
  • 图像增强、边缘检测、曲率、法线积分。
  • 误差评估、标定、对照实验。

3D与空间数据

  • SfM/MVS、点云、网格、正射图、纹理映射。
  • 局部RTI如何配准到整体3D模型。
  • IIIF、DeepZoom、瓦片和大图浏览。

AI标注与训练

  • YOLO bbox、分割mask、多边形、COCO/YOLO格式。
  • SAM点/框提示、自动预标注、人工审核。
  • 小样本、数据增强、训练/验证/测试划分。

文博与图像学

  • 汉画像石题材、构图、雕刻技法、拓片传统。
  • 前图像、图像志、图像学解释层级。
  • 证据链、研究者意见、版权与馆藏限制。

七阶段学习路线

Phase 1
2-3天

RTI/PTM基础

先理解RTI、PTM、H-RTI、HSH、反光球、光照方向和多光照图像集合MLIC。

  • 读：CHI术语表、H-RTI采集指南、Earl 2011系统综述、Malzbender 2001 PTM。
  • 做：画出采集到查看流程图；建立中英术语表。
  • 验收：能解释PTM和RTI的关系，知道为什么相机和对象必须固定。
Phase 2
4-7天

文化遗产案例

学习墓碑、楔形文字、岩画、硬币、安提凯希拉机械、碑铭学等案例，判断RTI适用边界。

  • 读：Mudge 2006、Palma 2010、Gravestone RTI、Epigraphy RTI、Archaeological PTM。
  • 做：整理“对象类型 -> RTI价值 -> 局限”对照表。
  • 验收：能判断哪些问题是采集问题，哪些是后处理问题。
Phase 3
1-2周

汉画像石与拓片

进入本项目核心对象：汉画像石、画像砖、碑刻、墓志、甲骨和浅刻文物。

  • 读：拓片恢复浮雕、汉画像石3D建模、河南汉画像石数据库、史语所拓本典藏。
  • 做：建立汉画像石对象类型、雕刻技法、输出类型和标注类别草案。
  • 验收：能说明“原石照片、拓片、RTI、3D模型、线图”各自解决什么问题。
Phase 4
2-4周

工具链与第一批实验

用公开样例和自采小样本跑通RelightLab/RTIViewer/OpenLIME，建立可复现实验记录。

  • 数据：CHI Fish Fossil、VAT楔形文字、硬币、刻痕板、石片。
  • 工具：RelightLab、RTIViewer、OpenLIME、ImageJ PS Tools。
  • 输出：RTI、增强图、法线图、虚拟掠射光、实验日志。
Phase 5
1-2月

固定光源矩阵与光度立体

这是最值得投入的中期主线。目标是从“能看清”进入“可复现、可比较、可生成法线/高度”。

  • 搭建16-32路LED矩阵，记录灯位和拍摄参数。
  • 读DiLiGenT-Pi、绘画光度立体、RTI profilometry。
  • 做刻痕标定板实验，比较RTI增强、法线图、高度图和人工线图。
Phase 6
1-2月

AI标注、线图和IIML

把图像增强结果变成可检索、可训练、可解释的数据。

  • YOLO先做伏羲女娲、乐舞人物、车马、神兽、建筑构件、铭文。
  • SAM/AnyLabeling/CVAT生成候选多边形，人工审核。
  • IIML保存资源版本、几何、语义、证据、置信度和研究者。
Phase 7
长期

前沿探索与平台化

在基础数据稳定后，再探索NeuralRTI、DisK-NeuralRTI、多光谱、偏振、RTI特征聚类和扩散线图。

  • NeuralRTI：先跑本地示例，再比较PTM/HSH/RBF/神经重光照。
  • 多光谱/偏振：先用低成本NIR/UV/偏振片做小样本。
  • Web平台：OpenLIME + IIIF + Annotorious/IIML。

仓库与工具选择

必须优先掌握

RelightLabRTIViewerOpenLIME

  • RelightLab/Relight CLI：新主工具，生成RTI、法线图、Web格式。
  • RTIViewer：经典查看器，用来理解增强模式。
  • OpenLIME：未来Web发布、标注和多层数据展示方向。

算法实验工具

NeuralRTIImageJ PSnear_ps

  • NeuralRTI：本地已下载，用于神经重光照学习。
  • ImageJ Photometric Stereo Tools：适合文物表面法线/高度实验。
  • near_ps：近场LED点光源光度立体，适合自建灯阵。

标注和训练工具

AnyLabelingCVATLabel Studio

  • AnyLabeling：最适合个人本地快速试用，支持YOLO、SAM、SAM2/3。
  • CVAT：正式项目/多人协作更强，格式支持全面。
  • Label Studio：适合把标注接入后端和模型预标注。

自研Web原型

IIIFAnnotoriousIIML

  • IIIF/DeepZoom解决大图和多版本资源发布。
  • Annotorious适合Web图像标注和OpenSeadragon集成。
  • IIML负责图像元素、几何、语义、证据链和知识图谱导出。

硬件选型与购买路线

硬件不要一步到顶。先用低预算包证明流程，再建设固定光源矩阵。真正影响结果的是稳定、光源方向、标定和元数据，而不是只买更贵相机。

低预算：学习验证型

3000-10000元

  • 已有相机或高质量手机。
  • 稳定三脚架、快门线、手持LED。
  • 黑色反光球、灰卡、色卡、比例尺。
  • 移动SSD和记录模板。

目标：跑通H-RTI、RelightLab/RTIBuilder、RTIViewer，输出增强图。

中预算：固定矩阵型

30000-80000元

  • RAW相机、微距镜头、翻拍架/重型三脚架。
  • 16-32路高显色LED和控制器。
  • 灯位标定、ArUco/棋盘格、刻痕标定板。
  • 32GB以上内存电脑和2TB以上SSD。

目标：固定灯位、逐灯拍摄、光度立体、法线图、数字拓片初稿。

高预算：科研平台型

150000元以上

  • 4500万-6000万像素相机或中画幅。
  • 48-96路可编程LED半球。
  • 偏振、多光谱、结构光/激光扫描。
  • NAS、工作站、暗室或移动采集架。

目标：RTI、光度立体、BRDF、多光谱、偏振、3D融合和Web交付。

推荐下单顺序

  1. 先买标定和稳定件：反光球、灰卡、色卡、比例尺、三脚架/翻拍架。
  2. 补微距能力：微距镜头或近摄方案。
  3. 搭建16路LED固定矩阵，先不要追求96路。
  4. 加入控制器和自动拍摄脚本，记录每次灯位和曝光。
  5. 实验稳定后再扩展到偏振、多光谱、3D扫描和NAS。

近期实验计划

实验A：官方样例闭环

  • 下载CHI Fish Fossil 1000px样例。
  • RelightLab输出PTM/HSH/RBF和法线图。
  • RTIViewer/OpenLIME查看增强模式。
  • 写实验日志：照片数、光照方向、输出格式、观察结果。

实验B：自采刻痕板

  • 用硬币、石片、手工刻痕板采集20-50张。
  • 输出增强图、法线图、虚拟斜光图。
  • 记录哪些角度能突出线刻，哪些产生伪影。
  • 形成第一版采集SOP。

实验C：固定光源矩阵

  • 16路LED，固定灯位，自动逐灯拍摄。
  • 保存lights.csv、capture.json和RAW。
  • 用光度立体估计法线/高度。
  • 和H-RTI手持光源结果对照。

实验D：YOLO/SAM/IIML小闭环

  • 选20-50张汉画像石公开图像。
  • 标注伏羲女娲、乐舞人物、车马、神兽等。
  • 用AnyLabeling或CVAT做bbox和polygon。
  • 导出YOLO/COCO，再转IIML JSON-LD。
