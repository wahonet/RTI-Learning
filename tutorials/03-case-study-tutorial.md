# 教程三：文化遗产案例怎么读

## 1. 为什么要读案例

RTI不是单纯的图像算法，它的价值来自具体对象：碑刻、墓碑、硬币、甲骨、岩画、浅浮雕。不同对象的材质、起伏、风化和研究目标不同，采集策略也不同。

读案例时不要只看结论，要重点看：
- 对象是什么。
- 传统方法有什么不足。
- RTI如何采集。
- 用了什么增强模式。
- RTI发现了什么。
- 哪些地方RTI没有解决。

## 2. 墓碑案例

阅读：`../papers/06_RTI_Gravestone_Detail.pdf`

重点：
- 风化文字。
- 苔藓和石材纹理干扰。
- 刻工细节和修正痕迹。

迁移到汉画像石：
- 风化和磨损问题相似。
- 文字和线刻都依赖低角度光照。
- 需要谨慎区分真实刻痕和材质噪声。

读完要回答：
- 墓碑案例中RTI解决了哪些传统照片无法解决的问题？
- 哪些增强模式最有用？
- 风化严重时RTI的局限是什么？

## 3. 岩画和野外H-RTI

阅读：`../papers/03_Mudge_2006_RTI_Rock_Art_Multiple_Viewpoint.pdf`

重点：
- 手持光源。
- 黑色反光球。
- 大型、不可移动、野外对象。
- 多视角RTI。

迁移到汉画像石：
- 馆藏小件可用固定装置。
- 摩崖、墓室、碑刻更接近野外H-RTI。
- 大对象可能需要分区采集和拼接。

读完要回答：
- H-RTI为什么适合野外？
- 反光球恢复光照方向的优势是什么？
- 多视角RTI对大型对象有什么价值？

## 4. 硬币案例

阅读：`../papers/04_Mudge_2005_RTI_Coins_Grand_St_Bernard.pdf`

重点：
- 小型高细节对象。
- 结构光和RTI互补。
- 交互重打光对纹饰和磨损的价值。

迁移到汉画像石：
- 硬币和汉画像石都涉及浅浮雕。
- 但汉画像石尺寸更大、材质更粗糙。
- RTI与3D扫描结合是后续方向。

读完要回答：
- RTI相比普通摄影记录了什么额外信息？
- 结构光/3D扫描和RTI如何互补？

## 5. 碑铭学案例

阅读：`../papers/07_RTI_in_Epigraphical_Studies.pdf`

重点：
- RTI作为记录和分析方法。
- 非接触、无损、可重复。
- 碑文识读和表面微地形。

迁移到汉画像石：
- 汉画像石不仅有文字，还有图像纹样。
- 标注和释读要与增强图绑定。
- 不能只存最终图片，必须保留原始数据和处理参数。

## 6. 案例摘要模板

每读一篇案例，按此模板写入对应阶段文档：

```markdown
## 文献标题

- 对象：
- 问题：
- 采集方法：
- 处理工具：
- 输出形式：
- RTI带来的新发现：
- 局限：
- 对汉画像石/碑刻的启发：
```

## 7. 本课输出

完成四篇案例后，整理：
- RTI适合的对象列表。
- RTI不适合或需要谨慎的对象列表。
- 汉画像石采集的优先局部：文字、线刻、风化边缘、刀痕、裂隙。
