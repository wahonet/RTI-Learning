# 英文资料怎么学：先中文化，再动手

这个领域绕不开英文资料。RTI、photometric stereo、IIIF、CIDOC CRM、YOLO 这些东西，中文资料有，但不系统。我的办法不是一上来硬读英文全文，而是先把资料拆开，翻译成自己能用的中文笔记。

## 总原则

先读能做事的部分，不追求一次读懂全部理论。

一篇英文论文或教程，我按这个顺序处理：

1. 翻译标题。
2. 翻译摘要。
3. 看图、表格、流程图和结论。
4. 找出设备、软件、参数、数据格式。
5. 把关键词做成中英文对照。
6. 写一页中文笔记。

如果一篇文章暂时看不懂，就先只留下三句话：

- 它研究什么对象。
- 它用了什么方法。
- 它对我的采集或结构化有什么用。

## 术语先背这些

### RTI 和成像

| English | 中文理解 |
| --- | --- |
| Reflectance Transformation Imaging, RTI | 反射变换成像，多方向打光后可以重新换光看表面 |
| raking light | 斜光，贴着表面照，用阴影显示浅刻 |
| highlight capture | 高光球采集，用反光球估计光源方向 |
| normal map | 法线图，记录表面朝向 |
| photometric stereo | 光度立体，用多光照估计表面形状 |
| relighting | 重新打光，在电脑里改变光照方向 |
| digital rubbing | 数字拓片，用图像或三维数据模拟拓片效果 |

### 三维采集

| English | 中文理解 |
| --- | --- |
| photogrammetry | 摄影测量，用多张照片重建三维 |
| SfM | 运动恢复结构，先算相机位置和稀疏点 |
| MVS | 多视图立体，生成稠密点云 |
| point cloud | 点云 |
| mesh | 网格模型 |
| texture | 纹理 |
| orthophoto | 正射影像 |

| laser scanning | 激光扫描 |

## 6 个月学习路线

### 第 1 个月：把 RTI 和斜光搞明白

任务：

- 读 RTI 入门资料。
- 翻译 RTI 采集指南里的关键步骤。
- 用硬币、石头或砖做一次 8-16 方向斜光拍摄。

成果：

- 一份中文 RTI 术语表。
- 一组自己的斜光实验照片。
- 一页实验记录：哪个角度最能看出细节。

### 第 2 个月：学会做一组可复核的采集

任务：

- 学曝光、白平衡、RAW、三脚架、比例尺、编号。
- 做一版采集表。
- 对一块碑或画像石局部拍多方向光。

成果：

- 一份采集 SOP 草稿。
- 一组可以交给别人复查的数据。

### 第 3 个月：摄影测量和三维基础

任务：

- 学 Meshroom 或 Metashape。
- 用 80-150 张照片重建一个小物件。
- 学会点云、网格、纹理、正射图这些输出。

成果：

- 一个小型 3D 模型。
- 一份“哪里重建失败”的记录。

### 第 4 个月：数字拓片和三维后处理

任务：

- 学 CloudCompare、MeshLab、GigaMesh 的基本查看。
- 尝试法线图、曲率图、环境遮蔽图。
- 理解数字拓片为什么比普通照片更适合读浅刻。

成果：

- 一张普通照片、斜光图、RTI 图、3D 渲染图的对比表。

### 第 5 个月：汉画像石结构化

任务：

- 建一个小词表：人物、马、车、乐器、建筑、日月、榜题。
- 用 VIA、Label Studio 或 CVAT 标注 50-100 张图。
- 记录场景、对象和关系。

成果：

- 一份 CSV/JSON 标注样例。
- 一份图像题材词表草稿。

### 第 6 个月：YOLO 和 AI 辅助

任务：

- 跑一次 Ultralytics YOLO。
- 只训练少量类别，比如人、马、车、榜题。
- 看漏检和误检，修改标注规范。

成果：

- 一个小模型。
- 一份错误分析。
- 一个“图像 -> 对象 -> 关系 -> 题材”的结构化样例。

## 每周安排

如果每周只有 6 小时：

- 2 小时翻译和读资料。
- 2 小时软件操作。
- 1 小时实拍或整理图片。
- 1 小时写中文笔记。

如果每周有 10 小时：

- 3 小时读资料和翻译。
- 3 小时跑软件。
- 2 小时采集实验。
- 2 小时整理成文档。

## 推荐的翻译方式

不要逐字翻全文。优先翻这些：

- 摘要
- 结论
- 方法流程
- 设备清单
- 参数表
- 图注
- 数据格式说明

最后写成自己的中文话：

```text
这篇文章解决什么问题：
它用了什么设备：
它的流程是什么：
我能借鉴什么：
我暂时没看懂什么：
```

这样读英文会慢一点，但资料会真正变成自己的东西。
