   - 总体阅读计划和阶段验收标准。

   - RTI、PTM、H-RTI、法线图和三维扫描区别。

3. [02-capture-processing-tutorial.md](02-capture-processing-tutorial.md)

4. [03-case-study-tutorial.md](03-case-study-tutorial.md)

6. [05-frontier-technologies-tutorial.md](05-frontier-technologies-tutorial.md)
   - 传统RTI之后的现代光度立体、Neural RTI、多光谱、偏振成像和AI线图。

   - 英文资料的中文化学习办法、术语表和 6 个月路线。

## 学习方式

- 先读教程，再读对应论文。
- 每篇论文写摘要，不需要一开始逐字翻译。
- 每个阶段都产出一个小文档或表格。
- 学完理论后立即用公开数据集做复现实验。
