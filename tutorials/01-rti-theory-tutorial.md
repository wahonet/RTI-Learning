# 教程一：RTI理论入门

## 1. RTI解决的核心问题

普通照片只记录一个光照条件下的文物外观。对于碑刻、甲骨、汉画像石这类对象，关键信息往往不是颜色，而是非常浅的凹凸、线刻、刀痕、磨损和裂隙。单张照片容易因为光照方向不合适而看不到这些信息。

RTI的思路是：固定相机和对象，从多个方向打光拍摄一组照片，再为每个像素建立“它在不同光照方向下会怎样变亮或变暗”的模型。这样后期就能在软件里移动虚拟光源。

## 2. RTI的基本流程

```text
固定相机和对象
  ↓
多方向打光拍摄照片序列
  ↓
估计每张照片的光照方向
  ↓
每个像素拟合反射模型
  ↓
交互重打光 / 增强显示 / 法线图 / 数字拓片
```

关键点：
- 相机不动，保证同一个像素始终对应文物表面的同一点。
- 光源移动，制造不同方向的阴影和高光。
- 光照方向必须记录或估计，否则无法建模。
- 输出不是简单拼图，而是可重打光的数据。

## 3. PTM是什么

PTM（Polynomial Texture Mapping）是RTI早期最重要的方法。它用一个二次多项式描述每个像素亮度与光照方向之间的关系：

```text
L = a0*lu^2 + a1*lv^2 + a2*lu*lv + a3*lu + a4*lv + a5
```

其中：
- `L`：某个像素在某个光照方向下的亮度。
- `lu, lv`：光照方向在图像平面上的投影。
- `a0-a5`：这个像素拟合出来的系数。

直观理解：每个像素都保存了一组“对光照变化的反应参数”。移动虚拟光源时，软件用这些参数重新计算这个像素的亮度。

## 4. H-RTI是什么

H-RTI是Highlight RTI。它不要求提前知道每个灯的位置，而是在画面里放一个反光球。每张照片中，光源会在球上形成一个高光点。软件根据高光点的位置反推光照方向。

优点：
- 低成本。
- 适合野外。
- 适合大型不可移动对象。

缺点：
- 反光球必须清晰可见。
- 高光点不能过曝。
- 反光球会占画面，需要后期裁掉。
- 光照分布依赖操作者经验。

## 5. RTI和三维扫描的区别

RTI不是传统三维扫描。

RTI擅长：
- 显示浅刻、刀痕、划痕、风化细节。
- 生成交互重打光图像。
- 提供表面法线和2.5D线索。

三维扫描擅长：
- 获取真实空间坐标。
- 测量尺寸和体积。
- 建立三维网格模型。

对汉画像石和碑刻，最理想的方向是二者结合：三维扫描保存整体形体，RTI保存局部微痕和表面信息。

## 6. 为什么RTI适合微痕扫描

微痕通常是极浅的凹凸变化。低角度光照时，浅凹凸会产生明显阴影；换方向照明，阴影也会变化。RTI记录了这些变化，所以能在后期放大微痕信息。

对公开微痕成像资料中的“不同入射角度光线”“光源矩阵”“微痕增强”来说，核心思想与RTI高度同源：都是通过多光照采集表面微小形态，再用算法增强。
